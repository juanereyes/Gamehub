About the site

Games first, secure foundations underneath.

GameHub is a home for browser-based versions of games I like. It is also a long-term portfolio project for demonstrating practical secure web development as I work toward a cybersecurity role.

Build approach

Separate the platform from the games.

General site work should stay focused on shared navigation, account flows, deployment, documentation, and security-sensitive architecture.

Each game can be built in its own focused pass, then integrated into the shared GameHub shell without turning the homepage into game-specific logic.
